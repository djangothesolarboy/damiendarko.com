Group Project 1 - Deadreads

Deadreads, a goodreads clone, is a social media application that allows users to review horror books, add horror books to bookcrypts along with, view reviews and profiles of other horror book lovers. Deadreads was made using Javascript, Express, and Pug.

Monster Dodge

Monster Dodge is a game I made using Unity and C# before beginning App Academy via a Udemy Course. It was the second game that I made using Unity. It is a simple game where you try to survive while dodging monsters falling from the sky. It was a fun project and a nice sort of intro to Unity. The art in the game was drawn digitally by me. Group Project 2 - 천재 Cheonjae

천재 Cheonjae (Genius) is a Genius.com clone but specifically for K-Pop songs! The application was made using a combination of Javascript, Flask, Python and PostgreSQL.

Features

  • Allows users to annotate songs.
  • Users can listen to soundcloud songs directly on site.
  • Users can watch YouTube videos directly on site.
  • Users are able to change their username.
  • Users can edit song information.
